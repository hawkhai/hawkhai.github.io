    原文：OpenCV documentation 4.0.0

    协议：CC BY-NC-SA 4.0

    欢迎任何人参与和完善：一个人可以走的很快，但是一群人却可以走的更远。

    1. OpenCV（开源计算机视觉库）是在 BSD 许可下发布的，因此它可以免费用于学术和商业用途。
    2. 它具有 C++，Python 和 Java 接口，支持 Windows，Linux，Mac OS，iOS 和 Android。
    3. OpenCV 专为提高计算效率而设计，专注于实时应用。该库以优化的 C/C++ 编写，可以利用多核处理。
    4. 通过 OpenCL 启用，它可以利用底层异构计算平台的硬件加速。
    6. OpenCV 在全球范围内采用，拥有超过 4.7 万用户社区，估计下载量超过 1400 万。
    7. 用途范围从交互式艺术，到地雷检查，网上拼接地图或高级机器人。
