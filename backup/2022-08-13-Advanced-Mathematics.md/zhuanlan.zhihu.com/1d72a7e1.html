在第六讲极限的运算法则中（戳我了解），我们知道，两个无穷小的和、差、乘积仍旧是无穷小，但是关于两个无穷小的商却出现不同的情况，例如，当 x→0x\to 0 时， 3x,x2,sin⁡x3x,x^2,\sin x 都是无穷小，而

limx→0x23x=limx→0x3=0limx→03xx2=limx→03x=∞limx→0sin⁡x3x=13limx→0sin⁡xx=13\lim\limits_{x\to 0}\frac{x^2}{3x}=\lim\limits_{x\to 0}\frac{x}{3}=0\qquad\lim\limits_{x\to 0}\frac{3x}{x^2}=\lim\limits_{x\to 0}\frac{3}{x}=\infty\qquad\lim\limits_{x\to 0}\frac{\sin x}{3x}=\frac{1}{3}\lim\limits_{x\to 0}\frac{\sin x}{x}=\frac{1}{3}

两个无穷小之比的极限反映了出不同的无穷小趋近于零的“快慢”程度。

假设在自变量同一个变化过程中，有 α,β\alpha,\beta 两个无穷小，且 α≠0\alpha\ne0 , limβα\lim\frac{\beta}{\alpha} 也是这个过程中的极限，有如下定义：

① 如果 limβα=0\lim\frac{\beta}{\alpha}=0 →β比α趋近0的速度快\xrightarrow{\beta比\alpha趋近0的速度快} β\beta 是比 α\alpha 高阶的无穷小，记作 β=o(α)\beta=o(\alpha) ；

② 如果 limβα=∞\lim\frac{\beta}{\alpha}=\infty→β比α趋近0的速度慢\xrightarrow{\beta比\alpha趋近0的速度慢}β\beta 是比 α\alpha 低阶的无穷小；

③ 如果 limβα=c≠0\lim\frac{\beta}{\alpha}=c\ne 0 ⇒\Rightarrow β\beta 与 α\alpha 是同阶无穷小；

④ 如果 limβαk=c≠0,k>0\lim\frac{\beta}{\alpha ^k}=c\ne 0,k>0 ⇒\Rightarrow β\beta 是关于 α\alpha 的 kk 阶无穷小；

⑤ 如果 limβα=1\lim\frac{\beta}{\alpha}=1 ⇒\Rightarrow β\beta β\beta 与 α\alpha 是等价无穷小，记作 α∼β\alpha \sim \beta 。

显然，等价无穷小是同阶无穷小的特殊情形，即 c=1 的情形。

下面举一些例子：

因为 \lim\limits_{x\to 0}\frac{3x^2}{x}=0 ，所以当 x\to 0 时， 3x^2 是 x 高阶的无穷小，即 3x^2=o(x)(x\to 0) 。

因为 \lim\limits_{n\to\infty}\frac{\frac{1}{n}}{\frac{1}{n^2}}=\infty ，所以当 n\to \infty 时， \frac{1}{n} 是比 \frac{1}{n^2} 低阶的无穷小。

因为 \lim\limits_{x\to 3}\frac{x^2-9}{x-3}=6 ，所以当 x\to 3 时， x^2-9 与 x-3 是同阶无穷小。

因为 \lim\limits_{x\to 0}\frac{1-\cos x}{x^2}=\frac{1}{2} (戳我了解），所以当 x\to 0 时， 1-\cos x 是关于 x 二阶无穷小。

因为 \lim\limits_{x\to 0}\frac{\sin x}{x}=1 ，所以当 x\to 0 时， \sin x 是与 x 的等价无穷小，即 \sin x \sim x(x\to 0) 。

还有一个常用的等价无穷小，如下面例 1 ，可直接作为结论使用：

例1. 证明：当 x\to 0 时， \sqrt[n]{1+x}-1\sim \frac{1}{n}x

证：证明二者是否为等价无穷小，只需证明商的极限为 1

\lim\limits_{x\to 0}\frac{\sqrt[n]{1+x}-1}{\frac{1}{n}x}=\lim\limits_{x\to 0}\frac{(1+x)^{\frac{1}{n}}-1}{\frac{1}{n}x}\xlongequal{分子分母同时乘以(1+x)^\frac{n-1}{n}+(1+x)^{\frac{n-2}{n}}+\cdots+(1+x)^{\frac{n-n}{n}}}\\\lim\limits_{x\to 0}\frac{[(1+x)^{\frac{1}{n}}-1 ]\cdot[(1+x)^\frac{n-1}{n}+(1+x)^\frac{n-2}{n}+\cdots+(1+x)^\frac{n-n}{n}]}{\frac{1}{n}x\cdot{\cdot[(1+x)^\frac{n-1}{n}+(1+x)^\frac{n-2}{n}+\cdots+(1+x)^\frac{n-n}{n}]} }\xlongequal{分子展开}\\\lim\limits_{x\to 0}\frac{[(1+x)^{\frac{1}{n}+\frac{n-1}{n}}+(1+x)^{\frac{1}{n}+\frac{n-2}{n}}+\cdots+(1+x)^{\frac{1}{n}+\frac{n-n}{n}}]-[(1+x)^{\frac{n-1}{n}}+(1+x)^\frac{n-2}{n}+\cdots+(1+x)^\frac{n-n}{n}]}{\frac{1}{n}x\cdot{\cdot[(1+x)^\frac{n-1}{n}+(1+x)^\frac{n-2}{n}+\cdots+(1+x)^\frac{n-n}{n}]}}\\=\lim\limits_{x\to 0}\frac{(1+x)^\frac{n}{n}+\cancel{(1+x)^\frac{n-1}{n}}+\cancel{\cdots\cdots}+\cancel{(1+x)^\frac{1}{n}}-[\cancel{(1+x)^\frac{n-1}{n}}+\cancel{(1+x)^\frac{n-2}{n}}+\cancel{\cdots\cdots}+(1+x)^\frac{n-n}{n}]}{\frac{1}{n}x\cdot{\cdot[(1+x)^\frac{n-1}{n}+(1+x)^\frac{n-2}{n}+\cdots+(1+x)^\frac{n-n}{n}]}}\\=\lim\limits_{x\to 0}\frac{(1+x)-1}{\frac{1}{n}x\cdot{\cdot[(1+x)^\frac{n-1}{n}+(1+x)^\frac{n-2}{n}+\cdots+(1+x)^\frac{n-n}{n}]}}=\lim\limits_{x\to 0}\frac{n}{{[(1+x)^\frac{n-1}{n}+(1+x)^\frac{n-2}{n}+\cdots+(1+x)^\frac{n-n}{n}]}}\\\xlongequal{分母每一项的值都趋近于1，共n项，所以分母也是n}\frac{n}{n}=1

定理1： \beta 与 \alpha 是等价无穷小的充分必要条件是 \beta=\alpha+o(\alpha) 。

证明：必要性：设 \alpha\sim \beta ，则 \lim\frac{\beta}{\alpha}=1 有

\lim\frac{\beta-\alpha}{\beta}=\lim(1-\frac{\alpha}{\beta})\xlongequal{极限差的运算法则}1-\lim\frac{\alpha}{\beta}=0

因此 \beta-\alpha=o(\alpha) ，即 \beta=\alpha+o(\alpha) 。

充分性：设 \lim \beta=\alpha+o(\alpha) ，则

\lim \frac{\beta}{\alpha}=\lim\frac{\alpha+o(\alpha)}{\alpha}=\lim(1+\frac{o(\alpha)}{\alpha})=1

因此 \alpha\sim \beta 。

例2.因为当 x\to 0 时， \sin x\sim x，\tan x\sim x，\arcsin x\sim x，1-\cos x\sim\frac{1}{2}x^2 ，所以当 x\to 0 时有 \sin x=x+o(x)，\tan x=x+o(x )，\arcsin x=x+o( x)，1-\cos x=\frac{1}{2}x^2+o(x^2)

定理2：设 \alpha\sim\alpha' ， \beta\sim\beta' ，且 \lim\frac{\beta'}{\alpha'} 存在，则 \lim\frac{\beta}{\alpha}=\lim\frac{\beta'}{\alpha'} 。

证明： \lim\frac{\beta}{\alpha}=\lim(\frac{\beta}{\beta'}\cdot\frac{\beta'}{\alpha'}\cdot\frac{\alpha'}{\alpha})=\lim\frac{\beta}{\beta'}\lim\frac{\beta'}{\alpha'}\lim\frac{\alpha'}{\alpha}=\lim\frac{\beta'}{\alpha'}

定理2表明，求两个无穷小之比的极限时，分子及分母可用等价无穷小代替，适当选择等价无穷小可简化计算。

例3.求 \lim\limits_{x\to 0}\frac{\tan 2x}{\sin 5x}

解： \lim\limits_{x\to 0}\frac{\tan 2x}{\sin 5x}\xlongequal[x\to 0时5x\to 0,\sin 5x\sim 5x]{x\to 0时2x\to 0,\tan 2x\sim 2x}\lim\limits_{x\to 0}\frac{2x}{5x}=\frac{2}{5}

例4.求 \lim\limits_{x\to 0}\frac{\sin x}{x^3+3x}

解： \lim\limits_{x\to 0}\frac{\sin x}{x^3+3x}\xlongequal[x\to 0时x^3+3x与它本身等价]{x\to 0时\sin x\sim x}\lim\limits_{x\to 0}\frac{x}{x(x^2+3)}=\lim\limits_{x\to 0}\frac{1}{x^2+3}=\frac{1}{3}

例5.求 \lim\limits_{x\to 0}\frac{(1+x^2)^\frac{1}{3}-1}{\cos x-1}

解： \lim\limits_{x\to 0}\frac{(1+x^2)^\frac{1}{3}-1}{\cos x-1}\xlongequal[x\to 0时，由例2，\cos x-1\sim -\frac{1}{2}x^2]{x\to 0时，x^2\to 0,由例1，(1+x^2)^\frac{1}{3}-1\sim \frac{1}{3}x^2}=\lim\limits_{x\to 0}\frac{\frac{1}{3}x^2}{-\frac{1}{2}x^2}=-\frac{2}{3}
