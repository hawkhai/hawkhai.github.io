先说反函数：

其实很简单

先把问题转换为标准的

y=f(x)y = f(x) 的形式

再用

x=f−1(y)x = f^{-1}(y)

这里 f−1f^{-1} 是 ff 对应的反函数形式，然后再把 x,yx,y 的符号交换一下次序就行了。

为了不引起混淆，你也可以用大写的 X,YX,Y 以示区分

下面给一个简单的例子来说明一下：

y＝sin12⁡3xy＝\sin^{\frac{1}{2}} 3x

两边平方：

y2＝sin⁡3xy^2＝\sin 3x

然后令： Y=y2,X=3xY= y^2, X = 3x ，这样标准形式是不是就出来了：

Y=sin⁡XY=\sin X

然后写出反函数：

X=arcsin⁡YX = \arcsin Y

再代回原来的符号：

3x=\arcsin y^2

整理一下：

x = \frac{1}{3}\arcsin y^2

最后交换次序得到反函数的结果：

y=\frac{1}{3}\arcsin x^2

由此可见求反函数其实一点也不难。

但上述结果并不是最终的完整结果，通常在写出反函数形式之后还必须要讨论反函数的定义域和值域！！！

由于原来的函数中包括 \sin^\frac{1}{2}3x ，因此 3x \in[2k\pi,\pi +2k\pi] （第一个半周期内非负，加减任意整数倍周期），对应的原来的函数的值域是 y\in[0,1]

根据你替换的过程，不难看出：

x \in[0,1]

y \in[\frac{2}{3}k\pi,\frac{\pi}{3} +\frac{2}{3}k\pi]

所以完整的结果应该是：

y=\frac{1}{3}\arcsin x^2

x \in[0,1]

y \in[\frac{2}{3}k\pi,\frac{\pi}{3} +\frac{2}{3}k\pi]

===================敲下重点================

一般问反函数求导的，通常都是需要这个--反三角函数求导：

1、反正弦函数y=\arcsin(x)的导数

\begin{align*} \left( \arcsin x \right)^\prime &= \frac{1}{{{\left( \sin y \right)^\prime }}} &&\color{Red}{(\frac{dx}{dy}=\frac{1}{\frac{dy}{dx}})} \\ & = \frac{1}{\cos y} && \color{Red}{(\cos y\ge0)}\\ &= \frac{1}{{\sqrt{1 – \sin^2 y} }} &&\color{Red}{(y=\arcsin(x))} \\ &= \frac{1}{{\sqrt{1 – {\sin^2}\left( {\arcsin x} \right)} }} &&\color{Red}{(\sin \left ( {\arcsin x} \right ) =x )} \\ &= \frac{1}{{\sqrt{1 – {x^2}}}} \end{align*} \\

2、反余弦函数 y=\arccos(x) 的导数

证法I： 类似推导

\begin{align*} \left( \arccos x \right)^\prime &= \frac{1}{{{\left( \cos y \right)^\prime }}} &&\color{Red}{(\frac{dx}{dy}=\frac{1}{\frac{dy}{dx}})} \\ & = -\frac{1}{\sin y} && \color{Red}{(\sin y\ge0)}\\ &= -\frac{1}{{\sqrt{1 – \cos^2 y} }} &&\color{Red}{(y=\arccos(x))} \\ &= \frac{1}{{\sqrt{1 – {\cos^2}\left( {\arccos x} \right)} }} &&\color{Red}{(\cos\left ( {\arccos x} \right ) =x )} \\ &= - \frac{1}{{\sqrt{1 – {x^2}}}} \end{align*} \\

证法II：由\arccos x =\frac{\pi}{2}-\arcsin x，于是

\left ( \arccos x \right )^\prime =\left ( \frac{\pi}{2}-\arcsin x \right ) ^\prime = - \left ( \arcsin x \right ) ^\prime =-\frac{1}{\sqrt{1-x^2}} \\

3、反正切函数 y=\arctan(x) 的导数

\begin{align*} \left( \arctan x \right)^\prime &= \frac{1}{{{\left( \tan y \right)^\prime }}} &&\color{Red}{(\frac{dx}{dy}=\frac{1}{\frac{dy}{dx}})} \\ & = \frac{1}{\frac{1}{\cos ^{2} y}} && \color{Red}{(\left( \tan y \right)^\prime=\sec^2x=\frac{1}{\cos ^{2} y})}\\ &= \frac{1}{1+\tan^2 y} &&\color{Red}{(\frac{1}{\cos ^{2} y}=\frac{\cos ^{2} y+\sin^{2} y}{\cos ^{2} y}=1+\tan^2 y)} \\ &= \frac{1}{{\sqrt{1 + {\tan^2}\left( {\arctan x} \right)} }} &&\color{Red}{(\tan\left ( {\arctan x} \right ) =x )} \\ &= \frac{1}{{{1 + x^2}}} \end{align*} \\

4、反余切函数 y=\operatorname{arccot} (x) 的导数

证法I：类似3，略。

证法II： 类似2，由y=\operatorname{arccot} (x)=\frac{\pi}{2}-\arctan(x) ，于是

\left ( \operatorname{arccot} x \right )^\prime =\left ( \frac{\pi}{2}-\arctan x \right ) ^\prime = - \left ( \arctan x \right ) ^\prime =-\frac{1}{1+x^2} \\

5、反正割函数 y=\operatorname{arcsec} (x) 的导数

\begin{align*} \left( \operatorname{arcsec} x \right)^\prime &= \frac{1}{{{\left( \sec y \right)^\prime }}} &&\color{Red}{(\frac{dx}{dy}=\frac{1}{\frac{dy}{dx}})} \\ & = \frac{1}{\tan y \cdot \sec y} && \color{Red}{(\left( \sec y \right)^\prime =\tan y \cdot \sec y)}\\ &= \frac{1}{|\sec y| \cdot \sqrt{\sec^2 y -1}} &&\color{Red}{(*)} \\ &= \frac{1}{|\sec \left ( {\operatorname{arcsec} x} \right ) |\cdot \sqrt{\sec^2 \left ( {\operatorname{arcsec} x} \right ) -1}} &&\color{Red}{(\sec \left ( {\operatorname{arcsec} x} \right ) =x )} \\ &= \frac{1}{|x| \sqrt{x^{2}-1}} \end{align*} \\

标 \color{Red}{(*)}部分主要是要把上一步完全由 \sec y 表示，由于有以下恒等关系
i) \ \sec^2 y - 1 = \frac{1}{\cos^2 y}-1 =\frac{1-\cos^2 y}{\cos^2 y}=\frac{\sin^2 y}{\cos^2 y}=\tan^2 y
因此: \tan y=\pm\sqrt{\sec^2 y -1}
ii) 这时必须注意到 y 的取值范围 $\left(0 \leq y<\frac{\pi}{2} \text { or } \frac{\pi}{2}见表1.）。而在这一步中不能取任何一个端点。同时注意到：
0 \leq y<\frac{\pi}{2} 时：\tan y,\sec y 都大于等于0
\frac{\pi}{2}<y \leq \pi 时：\tan y,\sec y 都小于等于0
因此： \tan y \cdot\sec y>0
综上：标\color{Red}{(*)} 步的写法可以保证这一不等关系始终成立。

6、反余割函数 y=\operatorname{arccsc} (x) 的导数

证法I：类似5，略。

证法II： 类似2，由y=\operatorname{arccsc} (x)=\frac{\pi}{2}-\operatorname{arcsec} x ，于是

\left ( \operatorname{arccsc} x \right )^\prime =\left ( \frac{\pi}{2}-\operatorname{arcsec} x \right ) ^\prime = - \left ( \operatorname{arcsec} x \right ) ^\prime =-\frac{1}{|x| \sqrt{x^{2}-1}} \\

注意：上述推导都没有严格说明其定义域

所以总结这俩表以备查用

反三角函数表\begin{array}{|c|c|c|c|} \hline \text { 反三角函数 } & \text { 三角函数} & \text {定义域} & 值域 \\ \hline y=\arcsin (x) & x=\sin (y) & -1 \leq x \leq 1 & -\frac{\pi}{2} \leq y \leq \frac{\pi}{2} \\ \hline y=\arccos (x) & x=\cos (y) & -1 \leq x \leq 1 & 0 \leq y \leq \pi \\ \hline y=\arctan (x) & x=\tan (y) & -\infty \le x +\infty & -\frac{\pi}{2}<y<\frac{\pi}{2} \\ \hline y=\operatorname{arccot}(x) & x=\cot (y) & -\infty \le x +\infty & 0<y<\pi \\ \hline y=\operatorname{arcsec}(x) & x=\sec (y) & x \leq-1 \text { or } 1 \leq x & 0 \leq y<\frac{\pi}{2} \text { or } \frac{\pi}{2}<y \leq \pi \\ \hline y=\operatorname{arccsc}(x) & x=\csc (y) & x \leq-1 \text { or } 1 \leq x & -\frac{\pi}{2} \leq y<0 \text { or } 0<y \leq \frac{\pi}{2} \\ \hline \end{array}

数学等 2 个话题下的优秀答主

常年有人点赞··· 你们这帮大一的小阿伟, 怎么那么喜欢反函数?

题主是大一吧？
那就仅直观理解, 不做严格的分析了.

其实这个问题很简单的：
理解『函数与反函数其实是同一条曲线』这句话就基本上就没问题了.

为了直观就举特例来讲:

构造 \[y=\sin x,x\in \left( -\frac{\pi }{2},\frac{\pi }{2} \right),\]
它的反函数就是 \[x=\arcsin y\] , 而没必要写成 \[y=\arcsin x.\]

只不过一个是 \[\left\{ y \right\}\] 向 \[\left\{ x \right\}\] 的映射, 一个是 \[\left\{ x \right\}\] 向 \[\left\{ y \right\}\] 的映射罢了, 然后映射的对应关系还是等价的. 实际上在数学中谁是自变量谁是因变量, 甚至谁是常数都是相对的. 你希望谁是啥谁就是啥了, 只要你自己心里有数别搞错就好. 定好俩量就可以求偏导数了, 如果其中一个是常量那导数就是零或者无穷咯, 当然了, 这在数学上来说并不太严谨[1].

首先, \[y=\sin x\ ,x=\arcsin y\] 画在笛卡尔坐标系里是同一条线.

你学的导数是极限定义的. 虽不严谨, 但用无穷小量定义的导数更直观, 在光滑函数上也不会出错.

y 关于 x 求导也就是 \[\frac{\text{d}y}{\text{d}x},\]

其中 {\rm d}x 可以认为是 Δx→0 的记号[2], 而 \[\text{d}y=\sin \left( x+\text{d}x \right)-\sin x.\]

所以显然, \[\frac{\text{d}y}{\text{d}x}\] 就是 y 在 x 处的变化率.

就当你已知 \[\frac{\text{d}y}{\text{d}x}=\cos x\] 吧.

那么他的反函数 \[x=\arcsin y\] 的导数自然就是 \[\frac{\text{d}x}{\text{d}y}\] , 也就是 x 在 y 处的变化率.

Well, 那直观来讲就有:

\[\left. \begin{align} & \frac{\text{d}y}{\text{d}x}=\cos x\Rightarrow \frac{\text{d}x}{\text{d}y}=\frac{1}{\cos x}=\frac{1}{\sqrt{1-{{\sin }^{2}}x}} \\ & \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ x=\arcsin y \\ \end{align} \right\}\Rightarrow \frac{\text{d}x}{\text{d}y}=\frac{1}{\sqrt{1-{{y}^{2}}}}.\]

而 \[x,y\] 就只是俩符号罢了, 你把 \[x,y\] 交换一下就是书上给出的公式了.

所以不要再被反函数这种概念迷惑了, 其实根本就不应该存在这么个多此一举的概念.

\[y=\sin x,x=\arcsin y\] 难道不都是一个意思?

甚至不必要写作 \[m=m\left( n \right)\] 的形式,

我是说 \[f\left( m \right)=g\left( n \right)\] 何尝不是确立了某种函数关系?

两边求微分一样能算导数.

反正重要的是等式内隐藏的映射关系.

所以说『反函数的导数是直接函数导数的倒数. 』

用极限的语言证明一下? 对性质良好的映射可以简单地认为有:

\[y=f\left( x \right)\Rightarrow x={{f}^{-1}}\left( y \right),\]

\[\frac{\text{d}y}{\text{d}x}\equiv \underset{\Delta x\to 0}{\mathop{\lim }}\,\frac{f\left( x \right)-f\left( {{x}_{0}} \right)}{x-{{x}_{0}}}\]

\[\ \ \ \ \ \ =\underset{\Delta x\to 0}{\mathop{\lim }}\,\frac{f\left[ {{f}^{-1}}\left( y \right) \right]-f\left[ {{f}^{-1}}\left( {{y}_{0}} \right) \right]}{{{f}^{-1}}\left( y \right)-{{f}^{-1}}\left( {{y}_{0}} \right)}\]

\[\ \ \ \ \ \ =\underset{\Delta x\to 0}{\mathop{\lim }}\,\frac{y-{{y}_{0}}}{{{f}^{-1}}\left( y \right)-{{f}^{-1}}\left( {{y}_{0}} \right)}\]

\[\ \ \ \ \ \ =\underset{\Delta x\to 0}{\mathop{\lim }}\,\frac{1}{\frac{{{f}^{-1}}\left( y \right)-{{f}^{-1}}\left( {{y}_{0}} \right)}{y-{{y}_{0}}}}=\underset{\Delta y\to 0}{\mathop{\lim }}\,\frac{1}{\frac{{{f}^{-1}}\left( y \right)-{{f}^{-1}}\left( {{y}_{0}} \right)}{y-{{y}_{0}}}}\equiv \frac{1}{\frac{\text{d}x}{\text{d}y}}.\]

为啥 \[\Delta x\to 0\] 可以换成 \[\Delta y\to 0\] 呢? 哈哈, 不告诉你.

很早之前用过一套教材，里面就没有反函数求导，而是把相关内容归到了隐函数部分，当时感觉处理得挺好的，而且也并没有影响到相关的内容，也没有影响系统的完整性。于是后来基本上都不讲反函数求导，而是通过其他途径解决相应的问题。比如反正弦反正切的导数是可以通过定义并由等价无穷小计算得到的。

个人建议，可以忽略反函数求导法。

里面所有的性质结论都可以来自隐函数。

而且反函数里面的各种关系还容易搞混。
