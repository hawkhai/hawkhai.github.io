  • 2022/05/12

本文内容

  1. 图像访问
  2. 图像完整性
  3. 图像修改

下面是 ImageHlp 函数。

图像访问

映像访问函数访问可执行映像中的数据。 这些函数提供对图像基础的高级访问，以及对图像数据最常见部分的非常具体的访问。

GetImageConfigInformation
GetImageUnusedHeaderBytes
ImageLoad
ImageUnload
MapAndLoad
SetImageConfigInformation
UnMapAndLoad

图像完整性

映像完整性函数管理映像文件中的证书集。

DigestFunction
ImageAddCertificate
ImageEnumerateCertificates
ImageGetCertificateData
ImageGetCertificateHeader
ImageGetDigestStream
ImageRemoveCertificate

图像修改

图像修改函数允许你更改可执行映像。

BindImage
BindImageEx
CheckSumMappedFile
MapFileAndCheckSum
ReBaseImage
ReBaseImage64
SplitSymbols
StatusRoutine
TouchFileTimes
UpdateDebugInfoFile
UpdateDebugInfoFileEx
