操做步骤

AS中，点击Help选择Edit Custom VM Options...

将Xmx数值调大，原来的很小。

原理分析

-xmx 参数是 Java 虚拟机启动时的参数，用于限制最大堆内存。

Android Studio 启动时设置了这个参数，并且默认值很小。

因为他不知道你电脑啥配置，就像不知道你酒量，默认你一杯醉。

如果你的电脑配置很好的话，那就调大。

结果验证

在 Settings -> Appearance 页里，打开 Show memory indicator 选项，然后主界面右下角会显示 Heap 总大小以及使用状况。这样操作一下，就可以确认 Heap Size 修改是否生效。

右下角展示，总共4012M，使用了854M。

风险披露

这是一个最基础的调法，适用于某些常规情况，还有很多高级的优化方式，需要具体结合你的项目来调整。
