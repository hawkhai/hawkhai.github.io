编译器警告 (级别 3) C4018

  • 项目
  • 2021/12/01

请为你的体验评分

反馈将发送到 Microsoft：按“提交”按钮，你的反馈将用于改进 Microsoft 产品和服务。 隐私策略。

谢谢。

本文内容

  1. 备注
  2. 示例
  3. 请参阅

'token' ：signed/unsigned mismatch

使用 标记 运算符比较 和 unsigned 数字需要编译器将值转换为 signedunsigned 。

备注

修复此警告的一种方式是在比较 和 类型时强制转换这两种类型 signed 之 unsigned 一。

示例

此示例生成 C4018，并演示如何修复此问题：

C++
// C4018.cpp
// compile with: cl /EHsc /W4 C4018.cpp
int main() {
    unsigned int uc = 0;
    int c = 0;
    unsigned int c2 = c; // implicit conversion

    if (uc < c)           // C4018
        uc = 0;

    if (uc < unsigned(c)) // OK
        uc = 0;

    if (uc < c2)          // Also OK
       uc = 0;
}
