什么是字符串？

字符串可以看作是特殊的数组，或者说是字符数组(数组的每个元素是一个字符)。在C++中，可以用字符串类型string来存储字符串。

字符串怎么用？

以下代码提供字符串常用的代码段：

//定义一个字符串
string p;
//定义一个字符串，并初始化为"abc"
string x("abc");
//连接字符串x和字符'a',并赋予y(执行完此句后y变成了"abca")
string y = x + 'a';
//从第1个字符开始，取出y的长度为2的子串(z的值为"bc")
string z = y.substr(1,2);

什么是字符？

在计算机中，用不同的数字(编码)来代表不同的字符。常用的编码有ascii, gbk2312, utf-8等等。而在C/C++语言中，一般的字符(包括英文字符，数字，英文符号等等)都用的是ascii编码。比如，我们可以对某个字符进行

//字符c等于ascii码为'a'(97)+9对应的字符(即'j')
char c = 'a'+ 9;

//字符c等于ascii码为106对应的字符(即'j')
char d = 106;

回文串

什么是回文串？

回文串，通俗地讲就是顺着读和倒着读一样的字符串。对于回文串s来说，它的第i位和倒数第i位是相同的，即s[i]==s[len(s)-1-i]。

数字与字符串间转换

如何将整形数转换成字符串？

在C/C++语言中，库函数itoa()可实现此功能，当然我们可以自己实现，整数转换成字符串的参考代码如下：

    string num2Str(long long num) {
        if (num == 0) {
            return "0";
        }
        string s, t;
        if (num < 0) {
            num = -num;
            t = "-";
        }
        while (num > 0) {
            s = (char)(num % 10 + '0') + s;
            num /= 10;
        }
        return t + s;
    }

如何将整形数转换成字符串？

在C/C++语言中，库函数atoi()可实现此功能，当然我们可以自己实现，字符串转换成整数的参考代码如下：

    int str2Num(string s) {
        int ret = 0, r = 0, neg = 1;
        if (s[0] == '-') {
            r = 1; neg = -1;
        }
        for (int i = r; i < s.length(); i++)
            ret = ret * 10 + s[i] - '0';
        return ret * neg;
    }

子序列

什么是子序列？

子序列定义为：由原数组或字符串中的部分元素，按照原有顺序组成的序列。（注意和子串区分开，子串的定义为在原字符串中连续的一串字符。所有的子串都是原串的子序列，子串在原串中是连续的，而子序列不一定。如字符串”123”，其所有的子序列为{“”,”1”,”2”,”3”,”12”,”13”,”23,”123”},所有子串为{“”,”1”,”2”,”3”,”12”,”23”,”123”}）

为何要高精度运算？

由于int能表示的整数范围大致在-21亿到21亿之间，若数字非常大(20位以上)，则在C与C++中只能用字符串表示，此时需要用高精度运算。在java语言中，有BigInteger类和BigDecimal类分别用于高精度整数运算和小数运算。

字符串匹配

字符串匹配任务中，我们需要查找模式串t是否在主串s中出现，若是，则求第一次出现的位置。

kmp算法

KMP算法用于中出现。其核心是利用匹配失败后的信息，尽量减少模式串与主串的匹配次数以达到快速匹配的目的。普通的匹配算法如下：

    int strStr(string s, string t) {
        if (t.empty()) {
            return 0;
        }
        if (s.empty()) {
            return -1;
        }
        int j = 0;
        for (int i = 0; i < s.length(); i++) {
            if (s[i] == t[j]) {
                j++;
                if (j == t.length()) {
                    return i - j + 1;
                }
            } else {
                i -= j; j = 0;
            }
        }
        return -1;
    }

在这个方法中，模式串与主串不匹配(s[i] != t[j])时，需要将t重新匹配，i左移到j位前，j从0重新开始。然而，我们并不需要从0开始匹配。我们可以预处理一个数组nxt[j]代表当j匹配失败时回退的位置。那么，nxt[j]应该为满足b[0:k]==b[j-k-1:j-1]最大的k。因为b[0:k]和b[j-k-1:j-1]相同，而b[0:j-1]和a[i-j+1:i-1]是匹配的。那么因为k<j-1，说明b[j-k-1:j-1]和a[i-k-1:i-1]是匹配的。即b[0:k]和a[i-k-1:i-1]匹配。因为k是求得的最大值，所以，对于任意k'>k，b[0:k']和a[i-k'-1:i-1]都不匹配。那么我们就b就不需要重新从0开始，只需要从k开始即可, 而数组a也不需要回退，只需要从i开始即可。

而求nxt时，相当于求b与自身子串的匹配。当b[i]与b[j]不匹配时，i不变，j回退到nxt[j]即可。

中心拓展法

中心拓展法用于求以每个字符及其间隙为中心的最长回文串。也可以用于字符串的最长回文子串。

在中心拓展法中，我们可以用特殊字符插入到原字符串中作为间隙。然后维护两个变量：mr与pos分别代表当前遍历过所有回文串到达的最右边的位置以及对应的中心,v[i]代表以i为。由于回文子串在对称轴两边对称，所以我们不必要每次都从长度为0开始搜索，可以首先找到i关于pos的对称点j=2pos-i。那么，因为子串s[i-min(v[j], mr-i): i+min(v[j],mr-i))]和s[j-min(v[j],mr-i):j+min(v[j],mr-i)]是对称的，且s[j-min(v[j],mr-i):j+min(v[j],mr-i)]是回文串，所以s[i-min(v[j], mr-i): i+min(v[j],mr-i))]也是回文串。那么我们只需要从长度为min(v[j],mr-i)开始搜索即可。
