1. 自编码器模型

这一讲我们将讲解2006年Geoffrey Hintton在Science上发表的文章Reducing the dimensionality of data with neural networks自编码器（Auto-Encoder）模型，基于这个模型，它部分地解决了神经网络参数初始化的问题。

2. 自编码器思想

自编码器采用的是分层初始化的思想。


例如我们想要训练图中 N N N层的神经网络，步骤如下：

步骤一：先训练图中下面这个小的网络，它的输入是原来的网络输入 X X X，中间只有一层，就是上面网络的第一层，而输出仍然是原网络的输出 X X X。


假设：
输入 X X X的维度是4维，第一层有3个神经元。上面这个小的网络可以详细的画成如下的形式


在这里可见原有的数据 X X X是4维，通过第一层Layer1后，4维数据被压缩为3维，而这3维的数据通过后面一层能够大致地恢复原有的4维数据X。换句话说，通过这个网络使中间的3维数据浓缩了原有的4维数据的信息，我们可以把它看作是对原有的4维数据的编码。又由于这是从 X X X到其自身的编码，因此，我们将这种算法叫作自编码器。

我们可以用后向传播算法来训练这个神经网络，当训练好第一层过后，我们下一个步骤是接着训练第二层。

步骤二：练好第一层过后，接着训练第二层。

假设第一层的输出为 X 1 X1 X1，在上面那个例子中， X 1 X1 X1是一个3维的向量。我们仍然采用自编码器的方法，将第二层的输出也设定为 X 1 X1 X1，然后用后向传播算法训练第二层，如下图所示。


注意： 这里后向传播算法仅仅作用在第二层，而第一层的参数是固定不动的。如果画出详细的图，将是如下形式：


尤其请大家注意的是在训练完第一层过后，会将上面那里左边的参数原封不动的拿下来作为第二层训练的基础。

步骤 M M M：以此类推，用同样的方法，在训练好前 M − 1 M-1 M−1层的前提下，我们可以用后向传播算法训练第 M M M层。


当完成所有N层的自编码器训练过后，我们将训练后获得的网络参数作为初始值，然后再次用后向传播算法调整整个网络的参数。


由于网络每一层初始值都不同程度的编码了训练样本的信息，因此，这样的初始化大概率保证了找到的局部极值点还是不错的。

它主要是基于前面章节所写的人工神经网络工具包，我们可以通过代码大致回顾一下自编码器的思想。例如在函数sae_create(SIZE) 中，输入和输出都是同一维度SIZE(1)，中间一层的维度是SIZE(2)，这是自编码其中对各层训练初始化的过程。而在函数 sae_train(sae,option,train_x) 中，可以看到，它直接调用了原来神经网络训练函数sae=nn_train(sae,option,train_x,train_x)，我们进一步观察可以发现，输入和输出都是同一个train_x，这也是我们前面讲到的用自编码器初始化每一层参数的标准步骤。

我们在手写体数据库Mnist上进行测试，在这个数据集上，我们设置的网络结构为 [784,400,169,49,10]，相当于总共有4层神经网络，激活函数Sigmoid，目标函数是交叉熵cross entropy，batch_size=100，iteration=2000。用同样的参数设置，我们使用自编码器训练的神经网络识别率为94.59%，而不使用自编码器训练的网络识别率为90.68%。可以看到，自编码器将识别性能提高了将近5个百分点，程序请参阅文后所附的MATLAB程序。

3. 结尾

在这一讲中，我们详细讲解了如何利用自编码器进行神经网络参数初始化的过程。最近几年来，自编码器作为一种特征提取器，其相关思想的应用非常广泛，例如最近流行的生成对抗网络（Generative Adversarial Networks）就部分的借鉴了自编码器思想。
