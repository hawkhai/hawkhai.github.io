Gradio 让你只需少量代码即可为机器学习模型构建和部署 web 应用程序。它的用途与 Streamlit 或 Flask 有交集，但更偏向快速搭建模型演示界面。

机器学习应用的几个框架：Gradio、Streamlit 和 Dash 进行简单的对比。

Gradio 的优势在于以下几点：
1. 允许进一步的模型验证。具体来说，可以用交互方式测试模型中的不同输入
2. 易于进行演示
3. 易于实现和分发，任何人都可以通过公共链接访问 web 应用程序。

据我所知，有两种解决方案：
* streamlit <https://streamlit.io/>
* Gradio <https://gradio.app/>

Gradio 应用可以通过 Hugging Face Spaces 托管并获得公开访问地址；免费额度、休眠策略和资源限制以 Hugging Face 平台为准。
