3.选择正确的解释，在后面画“✔”。
(1) 各种各样的鱼多得数不清。正像人们说的那样，西沙群岛的海里一段 半是水，一半是鱼。
A.这句话强调西沙群岛海里的鱼非常多，多到鱼和水各占一半的程度。
(   )
B.这句话强调西沙群岛海里的鱼非常多，但并不真是鱼和水各占一半。 这是一种夸张的写法。(   )
(2) 西沙群岛也是鸟的天下。
A.说明这里的鸟非常多。(    )
B.说明西沙群岛只有鸟类居住，这里真的是鸟的天下。(   )
