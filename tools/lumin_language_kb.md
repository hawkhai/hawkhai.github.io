# 路明语知识库

路明语（Lumin）是一门虚构的人造语言，设定为海岸城邦“黎湾”的通用贸易语。它的设计目标是短句清晰、词形稳定、便于在笔记、地图标注和简单对话中使用。

## 基本设定

- 中文名：路明语
- 自称：Lumin
- 使用地区：黎湾诸城、盐塔港、北星岛
- 书写方向：从左到右
- 常用文字：拉丁字母扩展拼写，不使用声调符号
- 语言风格：偏分析语，词序稳定，少量词缀表达时态和态度

## 发音规则

路明语的拼写基本按字母读音发音。

| 字母或组合 | 近似读音 | 说明 |
| --- | --- | --- |
| a | 啊 | 开口元音 |
| e | 欸 | 中前元音 |
| i | 衣 | 高前元音 |
| o | 哦 | 中后元音 |
| u | 乌 | 高后元音 |
| ai | 爱 | 双元音 |
| th | 清擦音 | 可近似读作“思” |
| zh | 浊擦音 | 可近似读作“日” |
| r | 轻颤音 | 可近似读作轻微卷舌 r |

重音通常落在倒数第二个音节上。例如 `lu-MIN`、`sa-VE-la`。

## 句法速查

路明语默认词序为：主语 + 动词 + 宾语。

示例：

```text
Mi lan sola.
我 看见 太阳。
```

形容词放在名词后：

```text
nava bren
船 大
```

否定词 `no` 放在动词前：

```text
Mi no lan sola.
我 没有看见太阳。
```

疑问句在句首加 `ka`：

```text
Ka tu vel mir?
你 要 水 吗？
```

## 时态与语气

路明语使用小词表达时间和语气，不改变动词本身。

| 小词 | 含义 | 例句 |
| --- | --- | --- |
| pa | 过去 | `Mi pa lan nava.` 我看见了船。 |
| nu | 现在 | `Mi nu vel mir.` 我现在想要水。 |
| fa | 将来 | `Mi fa rin.` 我将会出发。 |
| si | 可能 | `Tu si dom.` 你可能知道。 |
| da | 必须 | `Na da kor.` 我们必须工作。 |

## 构词规则

路明语可使用后缀构词。

| 后缀 | 含义 | 示例 |
| --- | --- | --- |
| -el | 工具或器物 | `kora` 手 -> `korael` 手工工具 |
| -in | 与某物相关的人 | `nava` 船 -> `navain` 船员 |
| -um | 地点 | `vale` 市集 -> `valeum` 市集区 |
| -ra | 抽象性质 | `mori` 安静 -> `morira` 宁静 |

## 小型对话

```text
A: Ka tu vel mir?
B: Mi vel mir, eso mi vel pelen.
A: Pelen ren oro.
B: Thala. Mi rin.
```

中文：

```text
A：你要水吗？
B：我要水，也要文书。
A：文书在门口。
B：谢谢。我出发了。
```

## 文化注记

路明语中 `lumin` 同时表示“光”和“道路”，因为黎湾传说认为海上道路由星光和灯塔共同标出。正式问候语 `Lumin ren tu` 字面意思是“光在你那里”，实际含义接近“愿你平安”。

## 语法扩展

### 名词复数

路明语名词默认不标记单复数，数量通常由数字或语境决定。需要明确复数时，在名词后加 `-i`。

| 单数或泛指 | 明确复数 | 中文 |
| --- | --- | --- |
| nava | navai | 船、船只们 |
| zhen | zheni | 朋友、朋友们 |
| kira | kirai | 星星、群星 |
| pelen | peleni | 文书、文书们 |

示例：

```text
Mi lan navai ren mir.
我看见许多船在水上。
```

### 所有格

所有关系用 `de` 表示，结构为“被拥有物 + de + 拥有者”。

```text
nava de mi
我的船
```

```text
zhen de tu pa rin.
你的朋友已经出发了。
```

### 比较级

比较结构使用 `mor`，格式为“甲 + 形容词 + mor + 乙”。最高级用 `mor-hal`。

```text
Nava mi bren mor nava tu.
我的船比你的船大。
```

```text
Kira nor mor-hal.
北星是最明亮的。
```

### 使役表达

使役小词为 `ma`，放在动词前，表示“使、让、令”。

```text
Vey ma nava rin.
风让船出发。
```

```text
Mi ma tari liti.
我让火变小。
```

## 语用规则

- 对长辈或港务官说话时，常在句尾加 `sa` 表示尊敬。
- 熟人之间可省略主语，但正式文书中不省略。
- 交易场景中，先说物品，再说数量，再说价格。
- 海上通信用语倾向短句，避免连续从句。

尊敬语示例：

```text
Mi vel pelen sa.
我需要文书，敬请办理。
```

交易语序示例：

```text
Thal sen du-dek mira.
盐五份，二十枚银币。
```

## 文书格式

黎湾港务文书常使用固定开头：

```text
Pelen de Valeum Lumin.
黎湾市集区文书。
```

货运登记常按“日期、船名、货物、数量、负责人”的顺序记录。

```text
Talun: nava Sola-Mir, thal sen-dek, navain Ravo.
今日：船“太阳水号”，盐五十份，船员 Ravo 负责。
```

## 迷你语料：港口告示

下面是一段路明语告示，可用于测试知识库系统的翻译、实体抽取和事实问答。

```text
Pelen de Valeum Lumin.
Talun prai, nava Sola-Mir rin ad Tora Thal.
Navain Ravo vel pelen-tama eso mir du barem.
Thalin Ema kor ren Valeum Ruda.
Oran si kran vesra, navai da rin mori.
Lumin ren tu.
```

中文参考译文：

```text
黎湾市集区文书。
今天早晨，太阳水号驶向盐塔港。
船员 Ravo 需要盖章文书和两桶水。
盐商 Ema 在红市集工作。
傍晚可能下雨，船只必须平静出发。
愿你平安。
```

可抽取事实：

- `Sola-Mir` 是一艘船。
- `Sola-Mir` 的目的地是 `Tora Thal`。
- `Ravo` 的职业是 `navain`，即船员。
- `Ravo` 需要 `pelen-tama` 和 `mir du barem`。
- `Ema` 是盐商，工作地点是 `Valeum Ruda`。
- 傍晚可能下雨。
